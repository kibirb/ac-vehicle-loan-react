# sp-call-centre-portal
AC react redux vehicle loan application.

## Instructions
 - $ git clone https://github.com/kibirb/ac-vehicle-loan-react.git
 - $ npm install
 
## Dev
 - $ npm run serve
 - Browse to http://localhost:9001/
 
## Test
 - $ npm run test
